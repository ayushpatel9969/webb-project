# Static Responsive Website - Onboarding Task

This is a foundational web project built as part of my company onboarding. It demonstrates core front-end development skills, including semantic structure, custom styling, and version control.

## 🚀 Tasks Completed
* **HTML5:** Created a semantic document structure.
* **CSS3:** Implemented custom fonts, colors, and layouts.
* **Responsive Design:** Added Media Queries to ensure compatibility across mobile and desktop devices.
* **Git Basics:** Managed the project using Git (Init, Add, Commit, Push).

## 🛠️ Tech Stack
* HTML5
* CSS3
* Git & GitHub

## 📱 Features
* **Mobile-First Approach:** Layout adjusts for screens smaller than 600px.
* **Clean Code:** Organized folder structure for assets and styles.

## 📂 How to View
1. Clone the repository.
2. Open `index.html` in any modern web browser.
